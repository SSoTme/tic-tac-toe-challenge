An implementation of 3x3 Tic Tac Toe is expected to support a number of different kinds of players. Human players, learning players and Rules based Players. Currently, there are 5 player types in this app.

  1. Human
    Represents a human player
    Human

  2. Beginner
    Wins, prevents a win or plays randomly.
    SimpleRules

  3. Intermediate
    This guy's pretty good - but not the best
    SimpleRules

  4. Advanced
    The best - all strategies are employed. This AI should never lose. It will only win if the opponent makes a mistake, but otherwise, it at least tie.
    SimpleRules

  5. God
